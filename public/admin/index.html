🛡️ Firestore Security Rules

Bu qaydaları Firebase Console → Firestore → Rules bölməsinə yapışdırın:

🔒 Tövsiyə Edilən Qaydalar (v5)

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Kandidatlar kolleksiyası
    match /candidates/{document} {

      // PUBLIC oxuma: yalnız status 'placed' olmayanlar
      allow read: if resource.data.status != 'placed';

      // ADMIN yazma/silmə: yalnız autentifikasiya + e-mail doğrulanmış
      allow write: if request.auth != null
                   && request.auth.token.email_verified == true;
    }

  }
}

🗄️ Firebase Storage Rules

Firebase Console → Storage → Rules bölməsinə yapışdırın:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {

    // CV şəkilləri — public oxuna bilər, yalnız admin yaza bilər
    match /cv-photos/{allPaths=**} {
      allow read:  if true;
      allow write: if request.auth != null
                   && request.auth.token.email_verified == true
                   && request.resource.size < 2 * 1024 * 1024
                   && request.resource.contentType.matches('image/.*');
    }

  }
}
