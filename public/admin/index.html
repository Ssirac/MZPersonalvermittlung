🛡️ Firestore Security Rules

Firebase Console → Firestore → Rules bölməsinə yapışdırın:

🔒 Tövsiyə Edilən Qaydalar (v5)

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /candidates/{document} {
      allow read: if resource.data.status != 'placed';
      allow write: if request.auth != null
                   && request.auth.token.email_verified == true;
    }
  }
}

🗄️ Firebase Storage Rules

Firebase Console → Storage → Rules bölməsinə yapışdırın:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /cv-photos/{allPaths=**} {
      allow read:  if true;
      allow write: if request.auth != null
                   && request.auth.token.email_verified == true
                   && request.resource.size < 2 * 1024 * 1024
                   && request.resource.contentType.matches('image/.*');
    }
  }
}
